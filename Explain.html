The blockquote element is used to indicate the quotation of a large section of text from another source. The diffrence between class and ID

  • A class selector is a full stop (“.”).
  • An ID selector is a hash character (“#”).


  • A class can be used to identify more than one element.
  • An ID can be used to identify one element only.


  • Same element could have more than one class.
  • Same element could not have more than one ID.


  • Class is not higly sensitive.
  • ID is higly sensitive.


  • Classes can overwrite each other i.e second class can overwrite the first class.
  • ID can overwrite the class.
